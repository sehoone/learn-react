# websquare-generator
### websquare 프로젝트를 react 로 전환하는 프로젝트에서 사용할 generator

websquare의 화면파일(xml)을 선택하여 data 부분을 typescript interface 파일로 변환하고 변환된 파일을 다운로드 받을수있는 generator project

주소: http://localhost:5173/xml-generator
